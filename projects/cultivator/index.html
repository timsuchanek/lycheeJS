    3. Plant your own Project

    Start a new project from the boilerplate by copying its contents into a new folder. 

    cd ~/lycheeJS/projects;
cp -R ./boilerplate ./myproject; # Replace myproject with unique identifier

    Modify the unique identifier of the lychee.Environment in the index.html, so that the project is now integrated with the remote debugging features and the lycheeJS Inspector. 

    var environment = new lychee.Environment({
	id: 'boilerplate', // Replace boilerplate with unique identifier
});

    Modify the lychee.net.Client's settings, so that it can connect to the lychee.net.Server.

    The API call should be replaced by an API call String (e.g. "/api/Server/?identifier=myproject") in order to be automatically integrated with the Sorbet WebSocket server balancer.

    Alternatively you can manually start your WebSocket server and/or use a fixed configuration for both the lychee.net.Server and lychee.net.Client (e.g. { host: 'myproject.lycheejs.org', port: 1337 }). 

    var settings = {
	client: '/api/Server/?identifier=boilerplate', // Replace boilerplate with unique identifier
};
